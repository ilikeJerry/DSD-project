# Human Factors (`docs/hf/`)

본 디렉터리는 **IEC 62366 사고의 Human Factors Engineering**을 “제품 운영 가능한 형태”로 구현하기 위한 검증·리뷰 근거이다.  
*(법적 Formative/Summative validation 산출물을 대체하지 않는다.)*

| 문서 | 용도 |
|------|------|
| `HFE_REVIEW_WORKFLOW.md` | 언제 HF 리뷰를 강제할지 |
| `HF_VALIDATION_CHECKLIST.md` | 출하/PR 게이트 체크리스트 |
